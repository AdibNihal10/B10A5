Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web.

How do you select an element from the DOM?

We can select an element from the DOM by different methods. We can use getElementById() to refer an id, getElementsByClassName to refer a class, getElementsByTagName to refer any tag, querySelector which will select the first element that matches a CSS selector and querySelectorAll which will select all elements that match a CSS selector (returns a NodeList).

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique that involves attaching a single event listener to a parent element to manage events triggered by its child elements, utilizing event bubbling. It's useful for improving performance by reducing memory usage, especially with dynamic content, and simplifying code maintenance by avoiding repetitive listeners for multiple child elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using element.setAttribute('attributeName', 'value') and styles using element.style.propertyName = 'value' in the DOM.
